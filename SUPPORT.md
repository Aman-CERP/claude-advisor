# Support

Second Opinion by AmanERP is a local open-source tool. AmanERP cannot inspect or recover your local runs, credentials, reports, or receipts.

## Installation and usage

Start with the [README](README.md), including prerequisites, installation, data flow, security posture, and the v0.1 migration path.

## Bugs and feature requests

Search [existing issues](https://github.com/Aman-CERP/amanerp-second-opinion/issues) before opening a report. Use synthetic data and include:

- plugin version;
- operating system and Python version;
- redacted `doctor` output;
- the exact command shape, without credentials or private paths;
- expected and observed behavior;
- a minimal safe reproduction.

## Security vulnerabilities

Follow [SECURITY.md](SECURITY.md) and use GitHub's private vulnerability-reporting flow. Never post a secret, private diff, or exploitable bypass in a public issue.

## Publisher and policy questions

For directory-listing, privacy, terms, trademark, accessibility, or other publisher questions, email `hello@amanerp.com`. Do not attach confidential source material or model transcripts.
