# Listing asset sources

These SVG files are the editable sources for the PNG screenshots shipped in the plugin manifest. They use AmanERP's established ivory, ink, teal, and gold palette and intentionally contain no Anthropic logos or claims of affiliation.

Export the two SVG files at their intrinsic `1600 × 1000` size when their content changes. `icon.png` and `logo.png` are square exports of the approved AmanERP logomark maintained by the AmanERP website project. Keep the final PNG files under `plugins/amanerp-second-opinion/assets/` so the release archive is self-contained and does not depend on a build-time renderer.
