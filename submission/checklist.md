# Publication readiness checklist

## Repository-controlled gates

- [x] Skills-only plugin with final file tree
- [x] Neutral AmanERP product identity
- [x] Public repository under Apache-2.0
- [x] Product, support, privacy, and terms URLs specified
- [x] Production icon, logo, and screenshots bundled
- [x] Three starter prompts
- [x] Exactly five positive test cases
- [x] Exactly three negative test cases
- [x] Credential-free reviewer fixture path
- [x] Optional live path uses only the reviewer's own Claude authentication
- [x] Release notes and policy record
- [x] Deterministic ZIP and SHA-256 generation
- [x] Local-only surface disclosure
- [x] Non-affiliation and trademark notice

## Human-controlled gates

- [ ] Website PR merged and the four public URLs verified live
- [ ] AmanERP business identity verified in OpenAI Platform
- [ ] Submitter has Apps Management write access
- [ ] Portal category confirmed from current available choices
- [ ] Selected countries revalidated at submission time
- [ ] Current portal attestations reviewed and accepted by an authorized AmanERP publisher
- [ ] Final ZIP, logo, screenshots, and test cases uploaded from the release commit
- [ ] OpenAI review approval received
- [ ] Authorized publisher selects Publish
