# Claude Advisor

Claude Advisor is a safety-first Codex plugin for obtaining an independent Claude opinion on bounded engineering decisions and pull requests.

Version 0.1.0 is under active development. The first release will provide explicit advisory and PR-review skills, durable provenance receipts, strict resource ceilings, and a no-tools Claude execution boundary.

See the [product and engineering specification](docs/specification.md) for the approved scope.
